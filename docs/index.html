Project Description

This tool allows you to reuse your old 6Dof device with serial connection (RS-232) in current versions of Windows OS (x86 and x64).


User guide [here]

Download [here]

Tested devices:

  • Spaceball 5000

Untested devices:

  • Spaceball 4000, 3000 and 2000
  • SpaceOrb 360
  • SpaceMouse

Let me know if there's another compatible device that works with this tool.
